Introduction to the gEVAL Browser

The gEVAL is the visual representation or the front end of a group of databases used to assist in sequencing and finishing genome projects of various vertebrate and non-vertebrate organisms.

Built using the framework, "pseudo-golden path", a term used to describe the minimum tiling path used in the assembly of the majority of the viewers. The closely associated term is AGP, in which we have AGP viewers for Human and mouse.

Supported browsers

The following browsers are fully supported, i.e. all site features including JavaScript and CSS styles should work properly:

  • Firefox 3+
  • Internet Explorer 7+
  • Safari 3+

We recommend Firefox for browsing Ensembl.

Other browsers

The following are not officially supported but should work with most or all of the site's current features:

  • Firefox 2
  • Camino
  • Opera 9+
  • Chrome
