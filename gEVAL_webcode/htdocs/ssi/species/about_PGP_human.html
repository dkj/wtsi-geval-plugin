Assembly

The human gEVAL viewer provides data set which is more current than GRCh37. The difference between this viewer and the human AGP viewer is the AGP viewer is based on the GRCh37, an assembly built on March 2009, whereas the gEVAL viewer is a more current snapshot. It also includes clones that are in the sequencing/finishing pipelines and have just been submitted (as of the date of this build).

The Genome Reference Consortium (GRC) maintains and improves the assembly.

A live status of the current tiling path can be found using the webtool Chromoview.

Manual annotations of the human genome can be found in Vega
