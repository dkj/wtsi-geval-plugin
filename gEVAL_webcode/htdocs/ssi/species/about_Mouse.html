The mouse gEVAL viewer provides data set which is more current than Build 38. The difference between this viewer and the mouse AGP viewer(s) is the AGP viewer is based on Build 38, an assembly built in early 2012, whereas the PGP viewer is a more current snapshot. It also includes clones that are in the sequencing/finishing pipelines and have just been submitted (as of the date of this build).

The Genome Reference Consortium (GRC) maintains and improves the assembly.

There are some major changes in the assembly from version m36; for more details see the NCBI build statistics .

A live status of the current tiling path can be found using the webtool Chromoview.

Manual annotation of this genome can be found in Vega
