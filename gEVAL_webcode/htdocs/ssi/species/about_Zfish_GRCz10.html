The zebrafish genome project is a collaboration between the Sanger Institute and the zebrafish community, announced during the Sanger Institute Zebrafish Workshop 2000 and was started in February 2001.

GRCz10 is the next iteration of the major release of the reference assembly of zebrafish, the last being Zv9 in 2010. Major chromosomal level changes and an additional 1500 sequence BAC and fosmid clones hve been included in this release. For more detailed information about this new assembly released by the GRC please visit the blog.

GRCz10 genome assembly record details can be found here: GCA_000002035.3.

The Genome Reference Consortium (GRC) maintains and improves the assembly.

A live status of the current tiling path can be found using the webtool Chromoview.

Manual annotation of this genome can be found in Vega
